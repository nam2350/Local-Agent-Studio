# Local Agent Studio

An interactive multi-agent platform for orchestrating local AI models. Build, visualize, and monitor collaborative agent networks using a modern node-based visual editor.

## Compatibility & Requirements

To ensure the best experience and performance, the following environment is recommended:

- **Node.js**: `v24`
- **Python**: `3.12+` (Backend & Model Execution)
- **Local AI Provider**: Compatible with `llama.cpp`, `Ollama`, or `vLLM` (OpenAI API Format expected).
- **Minimum VRAM**: 12GB+ (16GB+ recommended for running multiple 3B-8B local models simultaneously).
- **OS**: Windows 11 / Linux (Ubuntu) / macOS (Apple Silicon recommended).
